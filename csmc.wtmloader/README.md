================= [Intro] ============CSMC
wtmloader is a Comic Studio watermark     
changer. This can change any watermark    
text you want.                            
                                          
================= [Codes] ============CSMC
                                          
// by xMaxtris and Bang1338               
function wtmloader(wtm){                  
    cc.watermark = wtm;                   
}                                         
                                          
================= [Usage] ============CSMC
Open console, do this:                    
wtmloader("Custom watermark")             

Or use Tampermonkey script.
